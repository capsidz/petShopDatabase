1. Без индексов
```
EXPLAIN (ANALYZE, BUFFERS) 
SELECT * FROM pet WHERE age > 8;
```

С индексами
```
CREATE INDEX idx_pets_age ON pet(age);
```

```
EXPLAIN (ANALYZE, BUFFERS) 
SELECT * FROM pet WHERE age > 8;
```
